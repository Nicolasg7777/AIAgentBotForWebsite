Google Calendar Setup - Service Account (Simpler!)

Step 1: Create Google Cloud Project

  1. Go to Google Cloud Console
  2. Click "Select a project" ‚Üí "New Project"
  3. Name it "AI Support Agent Calendar" ‚Üí Click "Create"

Step 2: Enable Google Calendar API

  1. In the search bar, type "Google Calendar API"
  2. Click the API, then click "Enable"

Step 3: Create Service Account

  1. Go to "Credentials" in the left sidebar
  2. Click "Create Credentials" ‚Üí "Service Account"
  3. Name: "ai-support-calendar" ‚Üí Click "Create"
  4. Skip optional steps (click "Continue" then "Done")

Step 4: Download JSON Key

  1. Click on the service account you just created
  2. Go to "Keys" tab
  3. Click "Add Key" ‚Üí "Create new key"
  4. Choose "JSON" ‚Üí Click "Create"
  5. A JSON file will download - open it in a text editor

Step 5: Share Your Google Calendar

  1. Open Google Calendar
  2. Click the 3 dots next to your calendar ‚Üí "Settings and sharing"
  3. Scroll to "Share with specific people"
  4. Click "Add people" and paste the client_email from your JSON file
  5. Set permission to "Make changes to events" ‚Üí Click "Send"
